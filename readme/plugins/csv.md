# CSV Plugin Manual  
  
## Quick Overview  
  
The CSV plugin extracts translatable strings from standard CSV files.  
  
Each non-empty string is organized into a String Group. For context, the group name indicates the cell's row and column coordinates.  
Plugin settings can alterate this behavior.  
  
## pylivemaker  
  
The plugin is compatible with CSV files generated by [pylivemaker](https://github.com/pmrowla/pylivemaker).  
Refer to [livemaker.md](../engine/livemaker.md) for more infos.
