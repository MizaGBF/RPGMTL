# RPGMTL  
  
## Translating YU-RIS games  
  
Translating YU-RIS games relies on two plugins working together: [YBN](../plugins/ybn.md) and [YPF](../plugins/ypf.md).  
You can't import extracted `.ybn` files directly: The relevant OP Codes are computed during extraction from a `.ypf` archive.  
Translated `.ybn` can be used in a game by placing them in a directory named `ybin` in the game directory.
