# RPGMTL  
  
## Translating Unity games  
  
Unity Games are not supported by RPGMTL, due to how difficult it would be to extract and patch the strings.  
However, you can use the following workaround:
  
1. Apply [XUnity Auto Translator](https://github.com/bbepis/XUnity.AutoTranslator) to the game.  
2. Play through the game extensively to generate a comprehensive translation cache.  
3. Import the contents of the `_AutoGeneratedTranslations.txt` file into RPGMTL for manual editing and management alongside other supported file types.
